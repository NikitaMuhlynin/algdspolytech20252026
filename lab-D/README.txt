Вариант 27: задача о ферзях
Условие
На шахматной доске размером NxN расставить N ферзей, так, чтобы не какие два не биль друг друга.

Исходные данные
В текстовом файле input.txt записано число целое неотрицательное число N < 1000000.
Выведите в текстовый файл outputtxt координаты ферзей - номер ряда и номер колонки начиная нумерацию с 0.
